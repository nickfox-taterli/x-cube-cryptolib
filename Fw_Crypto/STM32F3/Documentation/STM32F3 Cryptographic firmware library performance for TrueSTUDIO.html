STM32F3 Cryptographic firmware library performance for TrueSTUDIO

Copyright 2015 STMicroelectronics


 

Introduction

This document provides a performance evaluation of the cryptographic library compiled with two optimization level (High speed and High size) for the STM32F3 microcontroller series.

The required code size by each algorithm is not provided in this document and it can be extracted from the map file after the building of project.

The tests were conducted on STM32F303 with CPU running at a frequency of 64 MHz and using the Atollic TrueSTUDIO STM32 (TrueSTUDIO) toolchain v5.3.0

1      AES results

1.1    AES symmetric key encryption algorithms performance

In this section we provide the performance results for:

       AES-128 in ECB, CBC, CTR, CFB, OFB, XTS and CMAC modes.

       AES-192 in ECB, CBC, CTR, CFB, OFB, XTS and CMAC modes.

       AES-256 in ECB, CBC, CTR, CFB, OFB, XTS and CMAC modes.

To calculate the number of cycles needed to perform each operation mode you can apply the below formula:

Clock Cycles = Init function + Append function * number of blocks of data (16bytes) + Finish function
