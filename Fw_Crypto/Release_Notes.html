STM32 Crypto Firmware Library is based on Cube package architecture package and includes a set of crypto algorithms based on firmware implementation ready to use in all STM32 series.

STM32 Crypto Firmware Library provides a rich set of examples covering the main features of each crypto algorithm. All examples are available in source file format and provided with preconfigured projects for the supported board. Expected output data are added in almost examples in order to check that the examples work correctly. Expected data are taken from crypto standard. Please refer to readme.txt in each example for more details about crypto standard used.



STM32 Crypto Firmware Library is located in middlewares level and support all STM32 series it is based on modular architecture that means new algorithms can be added without any impact on the current implementation. To provide flexibility for crypto functions, each algorithm can be compiled with different options to manage the memory and execution speed.

In Application layer, STM32 Crypto package provide a set of examples (up to 31 examples) covering all available algorithms with template projects for the most common development tools. Even without the appropriate hardware evaluation board, this layer allows you to rapidly get started with a brand new STM32 crypto firmware library.


STM32 Crypto Firmware Library Package update history


V3.1.0/ 30-October-2015

Main Changes
  • Maintenace release to update the examples projects.

V3.0.0/ 05-June-2015

Main Changes
  • Official release of STM32 Crypto Firmware Library V3.0.0 based on STM32 crypto library V2.0.6 available at ST website.
  • Support all STM32 series families without changing API vs last STM32 crypto library.
  • All API are based on firmware implementation without using any hardware acceleration.
  • New STM32 Crypto Libraries firmware compiled with GCC tool-chain.
  • STM32 Crypto Library is analyzed with CodeSonar tool .
  • New algorithms and modes added vs previous version 2.0.6 :
    • AES-128, AES-192, AES-256 bits :
      • FCB
      • OFB
      • XTS (Supports keys of 128 or 256 bits only)
    • CHACHA20
    • CHACHA20-POLY1305
    • Curve25519
    • ED25519
    • HASH functions with HMAC support:
      • SHA-384
      • SHA-512
    • POLY1305
    • RSA encryption/decryption with PKCS#1v1.5
