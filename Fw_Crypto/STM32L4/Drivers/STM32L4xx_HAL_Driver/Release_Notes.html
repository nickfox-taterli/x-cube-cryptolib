Update History

V0.5.0 / 10-February-2015

Main Changes

  • Fifth Beta release of STM32L4xx HAL Drivers dedicated to STM32L471xx/STM32L476xx/STM32L486xx devices.
  • The following HAL peripherals driver files have been updated :
    • FMC update: stm32l4xx_ll_fmmc.c
      • Enable write FIFO for Bank2/Bank3/Bank4 of FMC NOR/SRAM
    • HAL SPI update: stm32l4xx_hal_spi.c
      • Check BUSY flag at the end of HAL SPI processes

V0.4.0 / 06-February-2015

Main Changes

  • Fourth Beta release of STM32L4xx HAL Drivers dedicated to STM32L471xx/STM32L476xx/STM32L486xx devices.
  • STM32L471xx devices support added
  • The following HAL peripherals driver files have been updated in line with RM0351 V0.3 :
    • HAL RCC update: stm32l4xx_hal_rcc.h and extension model  stm32l4xx_hal_rcc_ex.h/.c
      • Functional change 
        • HAL_RCC_OscConfig() updates the Flash latency according to voltage range and configures the Systick when MSI range is updated and MSI oscillalator is used as the System clock source 
      • New API 
        • HAL_RCCEx_StandbyMSIRangeConfig() to set a MSI range after Standby 
      • Macros updated
        • __HAL_RCC_LSE_CONFIG() to avoid systematic disable of LSE
    • HAL PWR update: extension model stm32l4xx_hal_pwr_ex.h/.c
      • Functional change
        • HAL_PWREx_EnterSTOP1Mode() and HAL_PWREx_EnterSTOP2Mode() do not set any more LPR bit and prevent issues on wake-up
      • APIs renamed 
        • HAL_PWREx_ConfigVoltageScaling() renamed to HAL_PWREx_ControlVoltageScaling()
    • HAL COMP update: stm32l4xx_hal_comp.h
      • New macros 
        • __HAL_COMP_COMP1_EXTI_GENERATE_SWIT() and __HAL_COMP_COMP2_EXTI_GENERATE_SWIT()
      • Defines removed
        • COMP_MODE_LOWPOWER removed
    • HAL FIREWALL update: stm32l4xx_hal_firewall.h
      • Macros updated
        • __HAL_FIREWALL_PREARM_ENABLE(), __HAL_FIREWALL_PREARM_DISABLE(), __HAL_FIREWALL_VOLATILEDATA_SHARED_ENABLE(), __HAL_FIREWALL_VOLATILEDATA_SHARED_DISABLE(), __HAL_FIREWALL_VOLATILEDATA_EXECUTION_ENABLE() and __HAL_FIREWALL_VOLATILEDATA_EXECUTION_DISABLE()
    • HAL RTC update: extension model stm32l4xx_hal_rtc_ex.h
      • New macros 
        • __HAL_RTC_TAMPER_ENABLE_IT() and __HAL_RTC_TAMPER_DISABLE_IT()
    • HAL SAI update: stm32l4xx_hal_sai.h/.c
      • SAI_HandleTypeDef change
        • 'hdma' field replaced to two 'hdma_tx' and 'hdma_rx' fields                                                                     Application code impacted for __HAL_LINKDMA() operation.
      • SAI_InitTypeDef change
        • 'ClockSource' field removed as unused
      • APIs change
        • HAL_SAI_InitProtocol() takes uint32_t type as parameters instead of enum
    • HAL WWDG update: stm32l4xx_hal_wwdg.h
      • Macros updated 
        • __HAL_WWDG_ENABLE_IT() requires now a pointer on WWDG handle as parameter

V0.3.0 / 14-January-2015

Main Changes

  • Third Beta release of STM32L4xx HAL Drivers dedicated to STM32L4xx devices (STM32L476xx/STM32L486xx).
  • The following HAL driver have been added :
    • HAL FIREWALL: new files stm32l4xx_hal_firewall.h/.c
  • The following HAL peripherals driver files have been updated in line with RM0351 V0.3a :
    • HAL Generic: stm32l4xx_hal.h/.c
      • APIs renamed
        • HAL_SYSCFG_EnableVREFINT() renamed to HAL_SYSCFG_EnableVREFBUF()
        • HAL_SYSCFG_DisableVREFINT() renamed to HAL_SYSCFG_DisableVREFBUF()
        • HAL_SYSCFG_VREFINT_VoltageScalingConfig() renamed to HAL_SYSCFG_VREFBUF_VoltageScalingConfig()
        • HAL_SYSCFG_VREFINT_HighImpedanceConfig() renamed to HAL_SYSCFG_VREFBUF_HighImpedanceConfig()
        • HAL_SYSCFG_VREFINT_TrimmingConfig() renamed to HAL_SYSCFG_VREFBUF_TrimmingConfig()
      • APIs removed
        • HAL_SYSCFG_EnableExternalMemorySwapping() and HAL_SYSCFG_DisableExternalMemorySwapping()
    • HAL RCC update: stm32l4xx_hal_rcc.h/.c and extension model stm32l4xx_hal_rcc_ex.h/.c
      • New macros 
        • __HAL_RCC_SYSCLK_CONFIG() to configure the system clock source
      • Defines and macros renamed
        • RCC_PLL_USB1CLK renamed to RCC_PLL_48M1CLK
        • RCC_PLLSAI1_USB2CLK renamed to RCC_PLLSAI1_48M2CLK
        • RCC_FLAG_FIREWALLRST renamed to RCC_FLAG_FWRST
        • RCC_FLAG_PORRST renamed to RCC_FLAG_BORRST
      • API fixes
        • HAL_RCC_GetOscConfig() fixed to return the correct PLL multiplicator and divider factors
    • HAL COMP update: stm32l4xx_hal_comp.h/.c
      • Defines renamed
        • COMP_BLANKINGSRCE_TIM3OC5 and COMP_BLANKINGSRCE_TIM8OC3 corrected to COMP_BLANKINGSRCE_TIM3OC4 and COMP_BLANKINGSRCE_TIM8OC5
    • HAL CRC update: stm32l4xx_hal_crc.h
      • New macros
        • __HAL_CRC_SET_IDR() / __HAL_CRC_GET_IDR()
    • HAL CRYP update: stm32l4xx_hal_cryp.h/.c and extension model stm32l4xx_hal_cryp_ex.h/.c
      • CRYP_HandleTypeDef changes
        • Missing 'Instance' field added (shall be set to AES)
      • APIs changes
        • AES suspension/resumption functions HAL_CRYPEx_xxx() take now a pointer to the CRYP handle as parameter (CRYP_HandleTypeDef *hcryp)
    • HAL DAC update: stm32l4xx_hal_dac_ex.h
      • Defines removed
        • DAC_WAVE_NOISE and DAC_WAVE_TRIANGLE removed as unused in API
    • HAL FLASH update: stm32l4xx_hal_flash.h
      • New Defines
        • HAL_FLASH_ERROR_ECCD
        • FLASH_FLAG_ECCC (ECC correction flag) and FLASH_FLAG_ECCD (ECC detection flag)
    • HAL IRDA update: stm32l4xx_hal_irda.h
      • New macros
        • __HAL_IRDA_FLUSH_DRREGISTER()
        • __HAL_IRDA_CLEAR_FLAG() / __HAL_IRDA_CLEAR_PEFLAG() / __HAL_IRDA_CLEAR_FEFLAG() / __HAL_IRDA_CLEAR_NEFLAG() / __HAL_IRDA_CLEAR_OREFLAG() / __HAL_IRDA_CLEAR_IDLEFLAG()
        • __HAL_IRDA_ONE_BIT_SAMPLE_ENABLE() / __HAL_IRDA_ONE_BIT_SAMPLE_DISABLE()
    • HAL LPTIM update: stm32l4xx_hal_lptim.h/.c
      • Defines renamed
        • LPTIM_CLOCKSAMPLETIME_xxxTRANSISTIONS corrected to LPTIM_CLOCKSAMPLETIME_xxxTRANSITIONS
        • LPTIM_TRIGSAMPLETIME_xxxTRANSISTIONS corrected to LPTIM_TRIGSAMPLETIME_xxxTRANSITIONS
        • LPTIM_CLOCKPOLARITY_RISINGEDGE renamed to LPTIM_CLOCKPOLARITY_RISING
        • LPTIM_CLOCKPOLARITY_FALLINGEDGE renamed to LPTIM_CLOCKPOLARITY_FALLING
        • LPTIM_CLOCKPOLARITY_BOTHEDGES renamed to LPTIM_CLOCKPOLARITY_RISING_FALLING
    • HAL NAND update: stm32l4xx_hal_nand.h/.c
      • Typedef renamed
        • NAND_AddressTypedef corrected to NAND_AddressTypeDef
    • HAL NOR update: stm32l4xx_hal_nor.h/.c
      • Defines renamed
        • Enum NOR_StatusTypedef corrected to enum HAL_NOR_StatusTypeDef and values changed to HAL_NOR_STATUS_SUCCESS, HAL_NOR_STATUS_ONGOING, HAL_NOR_STATUS_ERROR and HAL_NOR_STATUS_TIMEOUT
      • API changed
        • HAL_NOR_GetStatus() returns a HAL_NOR_StatusTypeDef value
    • HAL PCD update: stm32l4xx_hal_pcd.h/.c and extension model stm32l4xx_hal_pcd_ex.h/.c
      • New APIs
        • Add support of low power mode (LPM): 
          • new PCD_LPM_StateTypeDef and PCD_LPM_MsgTypeDef
          • new API HAL_PCDEx_ActivateLPM(), HAL_PCDEx_DeActivateLPM() and HAL_PCDEx_LPM_Callback()
        • Add support of battery charging device (BCD):
          • new PCD_BCD_MsgTypeDef
          • new API HAL_PCDEx_ActivateBCD(), HAL_PCDEx_DeActivateBCD(), HAL_PCDEx_BCD_VBUSDetect() and HAL_PCDEx_BCD_Callback()
      • PCD_HandleTypeDef changes
        • 'LPM_State', 'BESL', 'lpm_active' and 'battery_charging_active' fields added for support of Low Power Mode (LPM) and Battery Charging Device (BCD)
    • HAL QSPI update: stm32l4xx_hal_qspi.h/.c
      • APIs changes
        • QSPI_CommandTypeDef field 'SDIOOMode' corrected to 'SIOOMode'
        • QSPI_SDIOO_INST_EVERY_CMD and QSPI_SDIOO_INST_ONLY_FIRST_CMD corrected to QSPI_SIOO_INST_EVERY_CMD and QSPI_SIOO_INST_ONLY_FIRST_CMD
    • HAL RTC update: stm32l4xx_hal_rtc_ex.h/.c
      • APIs removed
        • HAL_RTCEx_SetInternalTimeStamp_IT() and HAL_RTCEx_InternalTimeStampEventCallback()
    • HAL SMARTCARD update: stm32l4xx_hal_smartcard.h
      • New macros
        • __HAL_SMARTCARD_FLUSH_DRREGISTER()
        • __HAL_SMARTCARD_CLEAR_FLAG() / __HAL_SMARTCARD_CLEAR_PEFLAG() / __HAL_SMARTCARD_CLEAR_FEFLAG() / __HAL_SMARTCARD_CLEAR_NEFLAG() / __HAL_SMARTCARD_CLEAR_OREFLAG() / __HAL_SMARTCARD_CLEAR_IDLEFLAG()
        • __HAL_SMARTCARD_ONE_BIT_SAMPLE_ENABLE() / __HAL_SMARTCARD_ONE_BIT_SAMPLE_DISABLE()
    • HAL TIM update: stm32l4xx_hal_tim_ex.h/.c
      • APIs removed
        • HAL_TIM_SlaveConfigSynchronization_DMA()
    • HAL UART update: stm32l4xx_hal_uart.h
      • New macros
        • __HAL_UART_FLUSH_DRREGISTER()
        • __HAL_UART_CLEAR_FLAG() / __HAL_UART_CLEAR_PEFLAG() / __HAL_UART_CLEAR_FEFLAG() / __HAL_UART_CLEAR_NEFLAG() / __HAL_UART_CLEAR_OREFLAG() / __HAL_UART_CLEAR_IDLEFLAG()
        • __HAL_UART_ONE_BIT_SAMPLE_ENABLE() / __HAL_UART_ONE_BIT_SAMPLE_DISABLE()
    • HAL USART update: stm32l4xx_hal_usart.h/.c
      • USART_InitTypeDef changes
        • 'OverSampling' field removed to align with other STM32 series' HAL USART drivers
      • New macros
        • __HAL_USART_CLEAR_FLAG() / __HAL_USART_CLEAR_PEFLAG() / __HAL_USART_CLEAR_FEFLAG() / __HAL_USART_CLEAR_NEFLAG() / __HAL_USART_CLEAR_OREFLAG() / __HAL_USART_CLEAR_IDLEFLAG()
        • __HAL_USART_ONE_BIT_SAMPLE_ENABLE() / __HAL_USART_ONE_BIT_SAMPLE_DISABLE()

Known Limitations

  • MISRA-C 2004 compliance not finalized for all drivers
  • Code static analysis with CodeSonar not done
