Release Notes for STM32L4xx CMSIS

Update History

V0.4.0 / 06-February-2015

Main Changes

  • Fourth Beta version of CMSIS device files for ST STM32L4xx devices (STM32L471xx, STM32L476xx and STM32L486xx)
    • Add support of STM32L471xx devices (new files stm32l471xx.h and startup_stm32l471xx.s for IAR, ARM and GCC)
    • Updates according to RM0351 V0.3
      • RCC
        • RCC_CIFR_HSECSSF and RCC_CICR_HSECSSC definitions renamed to RCC_CIFR_CSSF and RCC_CICR_CSSC
        • RCC_CICR_LSECSSC added 

V0.3.0 / 14-January-2015

Main Changes

  • Third Beta version of CMSIS device files for ST STM32L4xx devices (STM32L476xx and STM32L486xx only)
    • stm32l476xx.h and stm32l486xx.h updates according to RM0351 V0.3a
      • FIREWALL
        • Bit description FIREWALL_xxx renamed to FW_xxx  
      • PWR
        • PWR_SR2_REGLPS definition corrected
      • SPI
        • IS_SPI_ALL_INSTANCE() assert macro corrected to check SPI3
      • SYSCFG
        • SYSCFG_MEMRMP_QFS definition removed
      • VREFBUF (renamed from VREF)
        • VREFBUF_TypeDef instead of VREF_TypeDef
        • VREFBUF_BASE address definition instead of VREF_BASE
        • Bit description VREF_CSR_xxx and VREF_CCR_xxx renamed to VREFBUF_CSR_xxx and VREFBUF_CCR_xxx
    • system_stm32l4xx.h
      • AHBPrescTable[] and MSIRangeTable[] declarations exported
    • GCC startup files startup_stm32l476xx.s and startup_stm32l486xx.s corrected for PVD_PVM_IRQHandler
    • Restore __ICFEDIT_region_RAM_start__ and __ICFEDIT_region_RAM_end__ symbols instead of __ICFEDIT_region_SRAM1_start__ and __ICFEDIT_region_SRAM1_end__ in IAR linker files stm32l476xx_flash.icf and stm32l486xx_flash.icf for IDE Linker configuration file edition

V0.2.0 / 28-November-2014

Main Changes

  • Second Beta version of CMSIS device files for ST STM32L4xx devices (STM32L476xx and STM32L486xx only)
    • stm32l4xx.h 
      • Add #define STM32L4xx
    • stm32l476xx.h and stm32l486xx.h updates according to RM0351 V0.2
      • FIREWALL
        • Added RESERVED1 and RESERVED2 FIREWALL_TypeDef 
      • EXTI
        • Updated to EXTI_IMR1/EXTI_IMR2_xxx, EXTI_EMR1/EXTI_EMR2_xxx, EXTI_RTSR1/EXTI_RTSR2_xxx,  EXTI_FTSR1/EXTI_FTSR2_xxx, EXTI_SWIER1_xxx/EXTI_SWIER2_xxx and EXTI_PR1_xxx/EXTI_PR2_xxx from EXTI_IMR_xxx, EXTI_EMR_xxx, EXTI_RTSR_xxx,  EXTI_FTSR_xxx, EXTI_SWIER_xxx and EXTI_PR_xxx
      • FMC
        • Add new base address for FMC_BASE, FMC_BANK1, FMC_BANK1_1, FMC_BANK1_2, FMC_BANK1_3, FMC_BANK1_4 and FMC_BANK3
        • Removed FMC_Bank2_TypeDef, FMC_Bank4_TypeDef and FMC_PIO4_xxx bit definitions
        • Updated Registers bit definitions to:
          • FMC_BCRx_xxx from FMC_BTR1_xxx, FMC_BTR2_xxx, FMC_BTR3_xxx and FMC_BTR4_xxx
          • FMC_BWTRx_xxx from FMC_BWTR1_xxx, FMC_BWTR2_xxx, FMC_BWTR3_xxx and FMC_BWTR4_xxx
          • FMC_PCR_xxx from FMC_PCR1_xxx, FMC_PCR2_xxx, FMC_PCR3_xxx and FMC_PCR4_xxx
          • FMC_SR_xxx from FMC_SR2_xxx, FMC_SR3_xxx and FMC_SR4_xxx
          • FMC_PMEM_xxx from FMC_PMEM2_xxx, FMC_PMEM3_xxx and FMC_PMEM4_xxx
          • FMC_PATT_xxx from FMC_PATT2_xxx, FMC_PATT3_xxx and FMC_PATT4_xxx
          • FMC_ECCR_ECC from FMC_ECCR2_ECC2 and FMC_ECCR3_ECC3
      • TIM
        • TIM_CCMR2_OC4MDMx corrected 

V0.1.0 / 06-October-2014

Main Changes

  • First Beta version (API freeze) of CMSIS device files for ST STM32L4xx devices (STM32L476xx and STM32L486xx only)
    • stm32l476xx.h and stm32l486xx.h updates according to RM0351 V0.2 (draft)
      • ADC
        • ADC1_2_COMMON_BASE renamed to ADC123_COMMON_BASE
      • CAN
        • CAN registers bit definitions completed
      • DFSDM
        • DFSDMx IRQ and peripheral base addresses changed (from DFSDM1..4 to DFSDM0..3)
        • DFSDM_Filter_TypeDef instead of DFSDM_TypeDef
      • OPAMP
        • Generic OPAMP_CSR_xxx, OPAMP_OTR_xxx and OPAMP_LPOTR_xxx bit definitions added
      • PWR
        • PWR_CR1_LPMS_SHUTDOWN bit added
      • RCC
        • RCC_BDCR_LSEDRV bit definition added
        • RCC_BDCR_LSECSSON and RCC_BDCR_LSECSSD bit definitions corrected
        • RCC_APB2ENR_FIREWALLEN renamed to RCC_APB2ENR_FWEN and value corrected
        • RCC_CSR_FIREWALLRSTF renamed to RCC_CSR_FWRSTF
      • SAI
        • SAI_xCR2_CPL bit definition added
      • SDMMC
        • SDMMC_CMD_ENCMDCOMPL, SDMMC_CMD_NIEN, SDMMC_CMD_CEATACMD, SDMMC_STA_CEATAEND,  SDMMC_ICR_CEATAENDC, SDMMC_MASK_STBITERRIE and SDMMC_MASK_CEATAENDIE bit definitions removed
      • USB registers description added
        • USB_OTG_GlobalTypeDef, USB_OTG_DeviceTypeDef, USB_OTG_INEndpointTypeDef, USB_OTG_OUTEndpointTypeDef, USB_OTG_HostTypeDef and USB_OTG_HostChannelTypeDef structures
        • Peripheral Base addresses (USB_OTG_FS_PERIPH_BASE, USB_OTG_GLOBAL_BASE, USB_OTG_HOST_BASE, USB_OTG_DEVICE_BASE, etc...)
      • VREF
        • VREF_CCR_TRIM instead of VREF_CRR_TRIM
    • IAR and ARM startup_stm32l476xx.s and startup_stm32l486xx.s updated for DFSDMx_IRQHandler updated from DFSDM1_IRQHandler..DFSDM4_IRQHandler to DFSDM0_IRQHandler..DFSDM3_IRQHandler
    • SRAM2_Region [0x10000000-0x10007FFF] declared in IAR linker files stm32l476xx_flash.icf and stm32l486xx_flash.icf
    • Template GCC startup_stm32l476xx.s and startup_stm32l486xx.s files added

V0.0.3 / 22-September-2014

Main Changes

  • Third Alpha version of CMSIS device files for ST STM32L4xx devices (STM32L476xx and STM32L486xx only)
    • stm32l476xx.h and stm32l486xx.h updates according to RM0351 V0.1Pub
      • VREF peripheral register description added
      • EXTI_TypeDef registers name changed
      • ADC, COMP and QUADSPI registers bit definitions updated
      • uint32_t definitions generalized for IWDG, TIM, WWDG

V0.0.2 / 01-August-2014

Main Changes

  • Second Alpha version of CMSIS device files for ST STM32L4xx devices (STM32L476xx and STM32L486xx only)
    • stm32l476xx.h and stm32l486xx.h updates according to RM0351 V0.1Pub
      • DBGMCU_TypeDef, RTC_TypeDef, SPI_TypeDef and SWPMI_TypeDef registers name changed
      • CRC_CR_POLSIZE renamed to CRC_CR_POLYSIZE
      • RCC_CR_CSSHSEON renamed to RCC_CR_CSSON
      • RCC_CSR_MSIRANGE renamed to RCC_CSR_MSISRANGE
      • RCC_CSR_PADRSTF renamed to RCC_CSR_PINRSTF
      • RTC_TAFCR bit definitions renamed to RTC_TAMPCR
      • USART_ISR_LBD renamed to USART_ISR_LBDF
      • DBGMCU_APB1L_FZ and DBGMCU_APB1H_FZ renamed to DBGMCU_APB1FZR1 and DBGMCU_APB1FZR2
      • IS_DMA_ALL_INSTANCE() debug macro now supports all DMA channels
    • startup_stm32l476xx.s corrected for RNG Global interrupt location in interrupt vector table
    • system_stm32l4xx.c updated for MSI frequency computation (SystemCoreClockUpdate(), new const table MSIRangeTable[12])

V0.0.1 / 11-July-2014

Main Changes

  • First Alpha version of CMSIS device files for ST STM32L4xx devices
    • Include files: stm32l4xx.h, stm32l476xx.h, stm32l486xx.h and system_stm32l4xx.h
    • Source files: system_stm32l4xx.c
    • Template Startup & memory scatter files:
      • IAR (EWARM toolchain): 
        • startup_stm32l476xx.s, stm32l476xx_flash.icf and stm32l476xx_sram.icf
        • startup_stm32l486xx.s, stm32l486xx_flash.icf and stm32l486xx_sram.icf
      • ARM (MDK-ARM toolchain): 
        • startup_stm32l476xx.s
        • startup_stm32l486xx.s
