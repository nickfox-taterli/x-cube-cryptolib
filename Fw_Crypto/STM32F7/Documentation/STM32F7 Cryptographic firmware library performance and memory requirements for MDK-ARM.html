STM32F7 Cryptographic firmware library performance and memory requirements for MDK-ARM

Copyright © 2015 STMicroelectronics


 

Introduction

This document provides a performance evaluation of the cryptographic library compiled with two optimization level (High speed and High size) for the STM32F7 microcontroller series.

The tests were conducted on STM32F756 with CPU running at a frequency of 84 MHz and using the RealView Microcontroller Development Kit (MDK-ARM) toolchain V5.14.

1      AES results

1.1    AES symmetric key encryption algorithms performance

In this section we provide the performance results and the required code size for:

·        AES-128 in ECB, CBC, CTR, CFB, OFB, XTS and CMAC modes.

·        AES-192 in ECB, CBC, CTR, CFB, OFB, XTS and CMAC modes.

·        AES-256 in ECB, CBC, CTR, CFB, OFB, XTS and CMAC modes.

To calculate the number of cycles needed to perform each operation mode you can apply the below formula:

Clock Cycles = Init function + Append function * number of blocks of data (16bytes) + Finish function
